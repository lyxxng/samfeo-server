# SAMFEO and SAMFEO++ Web Server
Web server for [SAMFEO](https://github.com/shanry/SAMFEO/tree/samfeo_prec#) & [SAMFEO++](https://github.com/shanry/FastDesign/tree/main#) RNA design.

SAMFEO paper (in *Proceedings of ISMB 2023*; journal version in *Bioinformatics*, 2023):
Tianshuo Zhou, Ning Dai, Sizhen Li, Max Ward, David H. Mathews, and Liang Huang (2023). [RNA Design via Structure-Aware Multi-Frontier Ensemble Optimization](https://academic.oup.com/bioinformatics/article/39/Supplement_1/i563/7210514?login=false).
